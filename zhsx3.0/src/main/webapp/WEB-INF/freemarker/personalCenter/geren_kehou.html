一、课后督课注意事项：【在听课评课的过程中，能灵活的运用看、听、想、问四种观察方法对本节课能够系统、专业、准确的给出评价。】

二、督课专业性要求：观察力是督课最重要的基本功之一；观察力之功力来之督课人员的长期督导实践的积累；即：督课走进课堂看什么？听什么？想什么？问什么？ 　　

关于看。不是简单的就听课而听，而是从专业、职业的视角对班级环境进行了全面的看，而且赋予思考性的分析，能透过表象发现问题，属于资深督课。

关于听。教师的三维目标落实是否有偏差；课堂鲜活的生成问题是否能及时予以应对。

关于想。没有此作基础，即便是发现了问题，也较难说准错在何处？

关于问。它是引发专业提问的前提，所以督课的提问很重要，但一定要建立在思考的基础上，问什么？何时问？
